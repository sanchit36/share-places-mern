# Share Places Mern

Share Places is a social media web app where a user can share places ( add image and something about it ). 

## Demo

To view a demo , **[click here](https://shareplace-ui.web.app/)**

## Technologies used

### Frontend
⚡️ Reactjs\
⚡️ React router\
⚡️ Mapbox\

### Backend
⚡️ Nodejs\
⚡️ Expressjs\
⚡️ Multer\
⚡️ Cloudnary\
⚡️ MongoDb and Mongoose\
